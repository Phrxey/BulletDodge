# Bullet Dodge

A bullet-hell dodge game centered around a bullet time mechanic. 
Control a spaceship, hold Shift to activate bullet time and weave 
through dense bullet patterns, then defeat the Boss to win.

## Controls
- WASD: Move
- C (hold): Slow movement
- Z: Fire
- Shift (hold): Activate bullet time (consumes energy bar)

## How to Run
Open the project in Unity 2022.3 LTS, load SampleScene, and press Play.
