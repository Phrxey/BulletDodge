# Thread The Needle

一个以子弹时间为核心机制的弹幕躲避游戏。玩家操控飞船，
按住Shift激活子弹时间穿梭密集弹幕，击败Boss即可获胜。

## 操作说明
- WASD：移动
- Z：发射子弹
- Shift（按住）：激活子弹时间（消耗能量槽）

## 如何运行
用Unity 2022.3 LTS打开项目，加载SampleScene场景，按Play运行。
